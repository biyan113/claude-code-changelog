# Claude Code Changelog 中文镜像

本仓库同步并翻译 [Claude Code](https://github.com/anthropics/claude-code) 的更新日志。

## 文件

- **CHANGELOG.md** - 英文原文（同步自 [anthropics/claude-code](https://github.com/anthropics/claude-code)）
- **CHANGELOG_CN.md** - 中文翻译

## 访问方式

### 最新更新
直接查看 [CHANGELOG_CN.md](./CHANGELOG_CN.md) 的顶部部分

### 全部更新
直接查看完整 [CHANGELOG_CN.md](./CHANGELOG_CN.md)

## 自动更新

- 每天 UTC 00:00 自动同步和翻译
- 通过 GitHub Actions 实现
- 支持手动触发工作流
